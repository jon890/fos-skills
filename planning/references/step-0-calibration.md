# Step 0 — 보정 (선택)

본격 질문 전, 질문의 깊이·강도를 사용자에 맞춘다. 매번 하지 않고 의도가 불명확할 때만.

## 맞출 것

- **지식 수준** (New / Working / Expert) — Expert 에게 기초 설명 낭비 금지, 학습자에게 과압박 금지. 설명 깊이 조정.
- **압박 강도** — Light(모호함만 해소) / Standard(가정에 도전) / Hard(실패 모드·롤백까지).

## 생략 기준

작은 변경이거나 사용자가 이미 방향을 명확히 줬으면 이 단계 생략. 생략도 announce 에 밝힌다("의도 명확 → 0단계 생략").

## 게이트 통과

수준·강도를 정했거나(또는 생략 사유를 밝혔고) → 1단계로.
